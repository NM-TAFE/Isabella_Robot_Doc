# Getting Started with ARI Robot

Welcome! This guide answers the most important question: **What can ARI do?**

## What is ARI?

ARI is an interactive robot that can **drive around your space, talk to people, and display presentations** on its integrated screen. It's designed to be easy to use while providing powerful capabilities for events, demonstrations, and interactive experiences.

## Core Capabilities

### 1. **Movement & Navigation** 🚀
- Drives smoothly in any direction
- Avoids obstacles autonomously
- Navigates spaces like offices, museums, events
- Can be controlled manually via phone or web interface

### 2. **Communication** 💬
- **Text-to-Speech (TTS)**: Speaks with clear, natural voice
- **Speech Recognition (ASR)**: Listens and understands voice commands
- **Facial Expressions**: Animated eyes show emotions
- **Audio**: High-quality speakers and microphone

### 3. **Display & Presentations** 🖥️
- Built-in touchscreen for interaction
- Shows slideshows, videos, information
- Displays web content
- Interactive interface for users

## What Can You Do with ARI?

### 🎤 **Event Host**
ARI can greet guests, explain exhibits, present information, and interact with audiences at conferences, museums, or trade shows.

**Real Example**: At a tech expo, ARI greets visitors at the booth, explains features, answers questions via voice, and displays slides on its screen.

### 🏫 **Educational Demonstrations**
Use ARI to teach concepts, engage students, run interactive lessons, or demonstrate robotics.

**Real Example**: A university uses ARI to teach AI and robotics fundamentals—students control it via apps and see real-time decision-making.

### 📸 **Media & Content Capture**
ARI can capture photos and video, then display them on its screen or in remotely located displays.

**Real Example**: At a conference, ARI takes selfies with attendees and displays them on a slideshow screen for entertainment and engagement.

### 🛡️ **Interactive Learning Games**
ARI can run educational games that teach cybersecurity, problem-solving, or other skills via voice and text interaction.

**Real Example**: Students play an interactive security game on ARI, learning about threats and defenses through progressive difficulty levels.

## Our Enhanced Features

We've added **three powerful features** to ARI that extend what it can do:

### 1. 📱 **Selfie-Cam Admin Control**
Control ARI from your phone—drive the robot, make it talk, take photos, and trigger presentations. Perfect for remote demonstrations or when you need hands-free operation.

**Use Case**: Run a demo where ARI drives through a crowd, stops at locations, talks to people, and takes photos—all from your phone.

👉 **Learn more**: [Selfie-Cam Simple Guide](./selfie-cam-simple.md)

### 2. 🖼️ **Remote Slideshow**
Display slideshows on a separate screen while ARI does other tasks. Perfect for events where you need a visual display in a different location.

**Use Case**: ARI circulates around an event taking photos, which automatically appear on a big screen elsewhere in real-time.

👉 **Learn more**: [Slideshow Simple Guide](./slideshow-simple.md)

### 3. 🎮 **Gandalf LLM Security Game**
Run an interactive cybersecurity learning game on ARI. Players try to convince an AI to reveal secrets, learning about social engineering and AI vulnerabilities.

**Use Case**: Educational workshop where ARI hosts a game teaching security professionals about prompt injection and AI safety.

👉 **Learn more**: [Gandalf Game Simple Guide](./gandalf-simple.md)

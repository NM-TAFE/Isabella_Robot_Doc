# Basic Features of ARI

ARI has three main capabilities. Let's explore what each one does and how you'll use it.

## 1. Movement & Navigation 🚀

### What It Does
ARI can move around autonomously or be controlled manually. It:
- Drives forward, backward, and turns in any direction
- Avoids obstacles using sensors
- Operates indoors and in open spaces
- Can be controlled via phone, web interface, or voice commands

### How You Use It

**Automatic Mode**: ARI explores and navigates on its own, avoiding obstacles
```
[Start] ──> [Scans environment] ──> [Plans route] ──> [Moves safely]
```

**Manual Mode**: You control ARI from a phone or computer
```
[You] ──(phone command)──> [ARI] ──> [Moves as directed]
```

### Real-World Scenarios

- **Museum Tour**: ARI moves through exhibits, guides visitors
- **Warehouse Demo**: ARI navigates through spaces autonomously
- **Event Coverage**: ARI circulates through crowds for social media
- **Research**: Collect data or observations from different locations

---

## 2. Communication 💬

### What It Does
ARI can talk and listen:
- **Speaks clearly** with text-to-speech (TTS) in multiple languages
- **Listens and understands** spoken commands (speech recognition)
- **Shows emotions** via animated eyes and facial expressions
- **Audio quality**: High-quality speakers and microphone

### How You Use It

**Speaking**:
```
You: "ARI, explain how you work"
ARI: [Processes request] → [Speaks: "I'm an interactive robot..."]
```

**Listening**:
```
Visitor: "Can you move to the left?"
ARI: [Recognizes voice] → [Moves left]
```

**Expressions**:
```
Happy greeting → [Animated smile]
Confused response → [Questioning expression]
Excited info → [Bright eyes]
```

### Real-World Scenarios

- **Customer Service**: ARI greets customers, answers FAQs
- **Education**: Teach lessons with voice explanations
- **Events**: Interactive Q&A with audience
- **Accessibility**: Voice commands for hands-free operation

---

## 3. Display & Presentations 🖥️

### What It Does
ARI has a built-in touchscreen that can:
- Display slideshows and photos
- Show videos and web content
- Display text, diagrams, and interactive elements
- Allow users to tap buttons and interact
- Present information in real-time

### How You Use It

**Standard Display**:
```
[Slides] → [Displayed on ARI's screen] → [Audience sees content]
```

**Interactive**:
```
[User taps button] → [ARI responds] → [Next slide appears]
```

**Content Types**:
- Slideshows (photos, presentations)
- Information dashboards (real-time data)
- Educational materials (diagrams, videos)
- Interactive forms (sign-ups, feedback)

### Real-World Scenarios

- **Product Demo**: Show features on ARI's screen while driving around
- **Digital Signage**: Display information as ARI moves through space
- **Guided Tours**: Step-by-step information displayed as you navigate
- **Engagement**: Interactive games or surveys on the screen

## Control Methods

You can interact with ARI in several ways:

### 1. **Phone Control** (📱 Easiest for events)
- Download app on smartphone
- Tap to move ARI around
- Speak to make ARI talk
- See what ARI's cameras see

### 2. **Web Interface** (💻 Easiest from desktop)
- Open web browser on any computer
- See interactive dashboard
- Control ARI via keyboard or mouse
- Monitor status and logs

### 3. **Voice Commands** (🎤 Hands-free)
- Speak naturally to ARI
- ARI responds to questions
- Request actions like "move forward" or "show slides"

### 4. **Touchscreen** (👆 Direct interaction)
- Touch ARI's screen directly
- Tap buttons and menus
- Interact with displayed content

## Key Features You'll Want to Know

### **Obstacle Avoidance**
ARI automatically detects and avoids obstacles—you don't need to worry about it running into things.

### **Battery Management**
ARI has a rechargeable battery. Check charge level before using, and charge after 2-3 hours of heavy use.

### **Web Interface**
Access ARI from any web browser on the same network. No special apps needed for basic control.

### **Emergency Stop**
There's always a way to immediately stop ARI if needed—safety first!

### **Networking**
ARI connects via WiFi to your network. Make sure your WiFi is strong and secure.

## Limitations (Good to Know)

- **Range**: ARI works best within range of your WiFi network
- **Battery**: Typical operation 2-3 hours per charge
- **Space**: Works best in open spaces; tight corners are tricky
- **Noise**: In very loud environments, voice recognition may need adjustment
- **Weather**: Indoor use only (not weatherproof)

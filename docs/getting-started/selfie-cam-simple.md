# Selfie-Cam Simple Guide

Control ARI from your phone. Drive it, make it talk, take photos.

## What is Selfie-Cam?

A smartphone app that lets you control ARI robot like a remote control. Instead of complicated menus, you get:
- 🎮 **Directional buttons** to move ARI
- 🎤 **Microphone** to make ARI speak
- 📸 **Camera button** to take photos
- 👁️ **Live view** from ARI's cameras

## Why Use It?

| Situation | Without Selfie-Cam | With Selfie-Cam |
|-----------|-------------------|-----------------|
| **Demo at event** | Need computer at ARI's location | Control from anywhere with phone |
| **Taking photos** | Awkward to position | Easy angle control |
| **Making ARI speak** | Pre-script everything | Talk live into phone |
| **Large venue** | Limited range from ARI | Work from audience area |

## Getting Started (30 seconds)

### 1. Download & Install
```
1. Download app to your smartphone
2. Connect to same WiFi as ARI
3. Open app
```

### 2. See What ARI Sees
- App shows live camera feed from ARI
- Real-time view helps you position ARI

### 3. Drive ARI
```
Tap: ↑ ↓ ← → buttons
ARI moves in that direction
```

### 4. Make ARI Speak
```
Type: "Hello everyone!"
Tap: Speak
ARI says: "Hello everyone!"
```

### 5. Capture Photos
```
Tap: Camera button
ARI takes photo
Photo automatically saves and uploads
```

## Core Features

### Movement Controls 🎮

**Simple Direction Buttons**
```
        ↑ (Forward)
    ← ↓ → (Left/Back/Right)
```

**How It Feels**
- Tap once = move a bit
- Hold button = continuous movement
- Release = stop

**Camera Feedback**
- See ARI's camera view while controlling
- Know exactly where ARI is heading

## Tips & Tricks

### 1. **Know Your WiFi Range**
- App works within WiFi coverage
- Test range before event
- Closer = faster response

### 2. **Use the Camera Feed**
- Always watch the live feed while driving
- Helps you anticipate obstacles
- Prevents accidental collisions

### 3. **Speak Clearly**
- ARI will repeat what you say
- Speak as if talking to person 10 meters away
- Avoid background noise

### 4. **Battery Awareness**
- Check ARI's battery before starting
- Heavy use: 2-3 hours per charge
- Keep charger nearby for longer events

### 5. **Pre-Script Important Lines**
- For key announcements, write first
- Avoid fumbling during important moments
- Type while ARI moves

### 6. **Test Before the Event**
```
1. Connect to WiFi
2. Drive ARI in a circle
3. Make it speak a test phrase
4. Take a test photo
5. Verify photo appears on slideshow
```

## Common Questions

### Q: Can others take control?
A: Yes! Multiple admins can connect. Great for team demonstrations.

### Q: What if WiFi drops?
A: ARI stops responding. It will hold position safely.

### Q: Can I use this outdoors?
A: Only if you have WiFi coverage outdoors.

### Q: How far away can I be?
A: Depends on WiFi strength. Usually 30-50 meters.

### Q: Is it secure?
A: Yes—secured by WiFi password. Only people on your network can control ARI.

### Q: What if ARI hits something?
A: It has sensors. Emergency stop available in app.

## Troubleshooting

### App won't connect
- ❌ ARI and phone on different WiFi? → Connect both to same network
- ❌ WiFi too far away? → Move closer to router
- ❌ App crashed? → Close and reopen

### ARI doesn't move
- ❌ Battery dead? → Charge ARI first
- ❌ Wrong network? → Check WiFi connection
- ❌ Obstacle in way? → Move it manually

### Photos not appearing
- ❌ Slideshow not running? → Start slideshow server first
- ❌ Auto-upload disabled? → Check settings
- ❌ Storage full? → Delete old photos
